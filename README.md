# MyCity Weather 🌤️  A Python-based weather application with two versions: 1. **API Key Version** – Fetches real-time weather data using OpenWeatherMap API. 2. **Latitude/Longitude Version** – Retrieves weather updates without requiring an API key, using coordinates for location.  ## ✨ Features - Current temperature, conditions, and wind speed - Two modes: API key or lat/long - Error handling for invalid inputs - Beginner-friendly code structure for learning API integration  ## 🚀 Usage 1. Clone the repo 2. Install dependencies: `pip install requests` 3. Run either version:    ```bash    python weather_with_key.py   # API key version    python weather_no_key.py     # Lat/long version
